# smart-water-level-monitoring
IOT project using ESP32 &amp; ThingSpeak to monitor water levels
## 📌 Objective
I developed a Smart Water Level Monitoring System using ESP32 and ThingSpeak. The system monitors water level data and sends it to the ThingSpeak cloud platform through WiFi. The data is displayed as real-time graphs, which helps in remote monitoring of water tanks.

## 🛠️ Components Used
- ESP32
- Ultrasonic Sensor (Water Level Sensor)
- ThingSpeak Cloud Platform

## ⚙️ Why ESP32?
ESP32 has built-in WiFi and is suitable for IoT applications.

## ☁️ Why ThingSpeak?
To store and visualize sensor data in the cloud.
